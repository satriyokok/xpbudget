# xpbudget
Expert system for budgetary
Adalah suatu aplikasi penyusunan anggaran
berdasar komponen2 anggaran.
Misalkan penyelenggaraan suatu Diklat.
Maka komponen nya adalah :
1. Jumlah peserta.
2. Jumlah Panitia.
3. Fasilitas Peserta :
   a. Seminar kit.
   b. Snack
   c. Makan
4. Narasumber / pengajar / trainer.
5. Rundown acara.
6. Tempat acara.
